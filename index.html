Página Web [Mapa de Navegação - Portal IFBA]

Objetivo: Criar um hub que centraliza os sites mais acessados do Portal IFBA, facilitando a navegação para estudantes e servidores, novatos ou veteranos, com foco em usabilidade e acessibilidade.

Tecnologias usadas: HTML, CSS e JavaScript.

Recursos implementados:

• Leitura Fácil: aumenta o tamanho de textos e botões para melhor legibilidade.
• Alto Contraste: modo de contraste elevado para melhor visualização.
• VLibras: integração com tradutor de Libras para acessibilidade a usuários surdos.

Resultado: Projeto aplicado na disciplina de Interação Humano-Computador, unindo conhecimentos de desenvolvimento web e boas práticas de acessibilidade.
